# SQL E-Commerce Business Analysis

## Project Overview
This project analyzes an e-commerce dataset using SQL to answer key business questions such as revenue trends, top customers, and product performance.

## Skills Demonstrated
- SQL Queries
- Data Aggregation
- GROUP BY
- HAVING
- Window Functions
- Business Data Analysis

## Dataset
The dataset contains order information including:
- Customer details
- Product category
- Quantity and price
- Order dates

## Business Questions Answered
- Total revenue generated
- Top selling products
- Revenue by category
- Top spending customers
- Monthly sales trends
- Orders distribution by city
